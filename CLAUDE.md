# 내 연금 계산기 — 프로젝트 명세서

## 프로젝트 개요

국민연금 / 퇴직연금(DC형) / 개인연금(IRP·연금저축) 세 가지 계좌에 월 얼마씩 납입하면
은퇴 후 월 수령액이 얼마나 되는지 계산해주는 단일 페이지 웹 앱.

**기술 스택**: Vite + React + TypeScript  
**스타일링**: Tailwind CSS  
**차트**: Chart.js (react-chartjs-2 래퍼)  
**상태관리**: useState / useReducer (외부 라이브러리 없이)  
**배포**: 단일 정적 파일 (GitHub Pages 또는 Vercel)

---

## UX 구조

### 탭 레이아웃 (2탭)

```
[ 입력 ]  [ 결과 ]
```

- **입력 탭**: 세 연금의 파라미터 입력 폼
- **결과 탭**: 계산 결과 + 그래프. 입력값이 바뀌면 실시간 반영.
- 입력 탭 우측 상단 "결과 보기 →" 버튼으로 결과 탭으로 이동.

## 입력 탭 상세

### 1. 국민연금 섹션

| 필드 | 타입 | 설명 |
|------|------|------|
| 현재 나이 | number input | 만 나이 |
| 총 납입 기간 | number input | 현재까지 납입한 개월 수 |
| 현재까지 납입 총액 | number input | 만원 단위 |
| 입력 모드 토글 | toggle | "월 소득으로 입력" ↔ "월 납입액 직접 입력" |
| 월 소득 (모드 A) | number input | 만원. 납입액 = 소득 × 9% 자동 계산 (직장인은 4.5% 본인부담 안내 문구 표시) |
| 월 납입액 (모드 B) | number input | 만원 직접 입력 |

**고정값 (UI에 안내 문구로 표시)**
- 수령 개시 나이: **65세 고정** (1969년생 이후 기준, 조기/연기 수령 없음)
- 적용 수익률: **국민연금공단 기금운용 실제 수익률 연평균 (2000~2023년 기준)**
  - 실제 값: 약 **5.92%** (코드에 상수로 박아두되, 출처 표기)
  - UI 하단에 "※ 2000~2023년 국민연금 기금운용 연평균 수익률 5.92% 적용" 표시

---

### 2. 퇴직연금 (DC형) 섹션

| 필드 | 타입 | 설명 |
|------|------|------|
| 현재 적립금 | number input | 만원 단위 |
| 월 납입액 | number input | 만원 단위 |
| 예상 연 수익률 | range slider + number | 1~15%, 기본값 5%, 0.1% 단위 |
| 은퇴 나이 | number input | 기본값 60 |

**고정값**
- 수령 방식: **연금형 고정** (일시금 옵션 없음)
- 수령 기간: 은퇴 나이 ~ 65세 사이 적립금이 퇴직연금으로 계속 운용되다가, 연금 개시 후 20년 수령 (또는 사용자 입력 — 아래 옵션 참고)
  - 단순화: 은퇴 시 적립금을 연금으로 전환, 수령 기간 동안 균등 분할 (이율 반영)

---

### 3. 개인연금 (IRP / 연금저축) 섹션

| 필드 | 타입 | 설명 |
|------|------|------|
| 현재 적립금 | number input | 만원 단위 |
| 월 납입액 | number input | 만원 단위 |
| 예상 연 수익률 | range slider + number | 1~15%, 기본값 5%, 0.1% 단위 |
| 연금 개시 나이 | number input | 기본값 65, 최소 55 |
| 수령 기간 | number input | 년 단위, 기본값 20 |

**세액공제 비교 (결과 탭에서 토글)**
- 입력 탭에는 토글 없음. 결과 탭의 개인연금 패널 상단에 토글 배치.
- **세액공제 받은 경우**: 수령 시 연금소득세 3.3~5.5% 원천징수
  - 나이별 세율: 70세 미만 5.5%, 70~80세 미만 4.4%, 80세 이상 3.3%
  - 간단화: 수령 기간 내 나이 기준으로 가중평균 세율 계산
- **세액공제 받지 않은 경우**: 납입 원금 부분은 비과세, 운용수익만 과세
  - 간단화: 전체 수령액 중 원금 비율만큼 비과세, 나머지에 세율 적용
- 두 경우의 월 수령액 차이를 강조 표시

## 결과 탭 상세

### 상단 요약 카드 (3개 나란히)

```
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│  국민연금    │  │  퇴직연금    │  │  개인연금    │
│  XX만원/월  │  │  XX만원/월  │  │  XX만원/월  │
└─────────────┘  └─────────────┘  └─────────────┘
         ↓ 합계 수령액 강조 표시 ↓
              총 XXX만원/월
```

### 그래프 1 — 적립금 성장 곡선 (꺾은선)

- X축: 현재 나이 ~ 은퇴 나이
- Y축: 적립금 (만원)
- 3개 라인: 국민연금(초록) / 퇴직연금(파랑) / 개인연금(주황)
- 은퇴 시점 수직선 표시 (dashed)

### 그래프 2 — 연간 수령액 막대그래프 (은퇴 후)

- X축: 은퇴 나이 ~ 수령 종료 나이 (또는 90세까지)
- Y축: 연간 수령액 (만원)
- Stacked bar: 국민연금 / 퇴직연금 / 개인연금 3색 누적

### 개인연금 세금 비교 패널

```
[ 세액공제 받은 경우 ○ ]  ←→  토글  →  [ 세액공제 안 받은 경우 ]

세액공제 O: XX만원/월   |   세액공제 X: XX만원/월
연금소득세 적용         |   원금 비과세
차이: +X만원/월 (세액공제 X가 유리 / 세액공제 O가 유리)
```

## 계산 로직 명세

### 국민연금 예상 수령액

국민연금 수령액은 공식 산식(A값 + B값)이 복잡하므로, **단순화된 추정 방식** 사용:

```
은퇴 시 총 납입 기간(개월) = 현재까지 납입 기간 + (65 - 현재나이) × 12
은퇴 시 예상 적립금 = 현재 납입총액 × (1 + r)^n + 월납입액 × [((1+r)^n - 1) / r]
  - r = 월 수익률 = 5.92% / 12
  - n = 남은 납입 개월수 = (65 - 현재나이) × 12

월 수령액 추정 = 은퇴 시 적립금 / (예상 수령 기간 개월수)
  - 예상 수령 기간: 기대수명 83세 기준 → 65세부터 18년 = 216개월
  - (이율 적용 연금 분할 공식 사용: PMT 공식)
```

> 참고: 실제 국민연금은 A값(전체 가입자 평균소득)에 연동되므로 정확한 계산은 불가. 적립금 기반 추정임을 UI에 명시.

### PMT 공식 (연금 월 수령액)

```typescript
function calcPMT(rate: number, nper: number, pv: number): number {
  // rate: 월 이율, nper: 수령 개월수, pv: 은퇴 시 적립금
  if (rate === 0) return pv / nper;
  return (pv * rate * Math.pow(1 + rate, nper)) / (Math.pow(1 + rate, nper) - 1);
}
```

### 퇴직연금 월 수령액

```
은퇴 시 적립금 = FV(현재 적립금, 월 납입액, 수익률, 은퇴까지 개월수)
월 수령액 = PMT(수익률 / 12, 수령기간 × 12, 은퇴 시 적립금)
수령 기간 기본값: 20년
```

### 개인연금 월 수령액

```
연금 개시 시 적립금 = FV(현재 적립금, 월 납입액, 수익률, 개시까지 개월수)
세전 월 수령액 = PMT(수익률 / 12, 수령기간 × 12, 적립금)

세액공제 O → 세후 수령액 = 세전 × (1 - 실효세율)
  실효세율: 수령 기간 내 나이별 가중평균
    - 개시나이 ~ 70세: 5.5%
    - 70세 ~ 80세: 4.4%
    - 80세 이상: 3.3%

세액공제 X → 원금 비율 = 총 납입 원금 / 연금 개시 시 적립금
  세후 수령액 = 세전 × (원금비율 + (1 - 원금비율) × (1 - 실효세율))
```

## 디자인 가이드

- **컬러**: 국민연금 = 초록(#1D9E75), 퇴직연금 = 파랑(#378ADD), 개인연금 = 주황(#EF9F27)
- **폰트**: Pretendard (CDN)
- **입력 필드**: 오른쪽 정렬, 단위(만원, %, 세, 년) suffix 표시
- **슬라이더**: 수익률 입력은 슬라이더 + 숫자 input 연동
- **반응형**: 모바일(~768px)에서도 탭 레이아웃 유지, 카드 1열로 전환
- **결과 탭**: 입력값이 없으면 "—" 표시, 있으면 실시간 계산
